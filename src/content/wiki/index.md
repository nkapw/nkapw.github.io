This wiki collects working notes on how to capture, connect, and keep ideas
useful over time. It is deliberately plain: text, links, and a little structure.
Nothing here depends on a particular app — the methods outlive the tools.

## Start here

If you are new to note-based thinking, read these in order. The links below use
`[[wikilink]]` syntax — the plain way to connect notes in this wiki:

1. **[[zettelkasten|Zettelkasten]]** — the slip-box method that started
   most modern note systems.
2. **[[evergreen-notes|Evergreen notes]]** — notes written to be reused
   and developed, not filed away.
3. **[[linking-your-thinking|Linking your thinking]]** — the atomic-note
   philosophy that ties the others together.
4. **[[tools|Tools]]** — a plain comparison of software that supports
   these practices.

## The one idea

If you take away a single thing, take this: **a note is only valuable if you
can find it again and build on it.** Filing is easy; the hard part is making
your past thinking show up exactly when a new idea needs it. Every method in
this wiki is, at bottom, a different answer to that one problem.

Write notes for the version of you who has forgotten everything. Link
generously. Revisit often.
